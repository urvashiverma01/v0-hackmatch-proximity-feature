## 💡 The Core Idea
* **The Problem:** Team formation at hackathons is chaotic, leading to unbalanced skillsets and team friction.
* **The Solution:** An AI engine that analyzes user profiles (tech stack, personality, availability) to automatically match solo hackers and fill team skill gaps.

## ✨ Features Built
* **AI Matchmaking:** Auto-pairs users based on complementary skills and project goals.
* **Synergy Assessment:** Filters matches by work style and personality compatibility.
* **Live Dashboard:** Clean UI to browse recommended teammates in real-time.

## 🛠️ Tech Stack
* **Frontend & Core Logic:** Next.js, React, Tailwind CSS, Shadcn UI (Built using v0 by Vercel)
* **State Management & Matching:** Client-side React State (Mock AI Matchmaking data optimized for prototype demonstration)
